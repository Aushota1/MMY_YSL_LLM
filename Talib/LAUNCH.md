# Запуск и команды — Talib

Краткое описание запуска приложения и полезных команд. Подробности по установке — в [README.md](README.md).

---

## Требования

- Python 3.8+
- Установленные зависимости из `requirements.txt`

---

## Установка зависимостей

Выполнять из корня проекта **tokenier** или из папки **Talib**:

```bash
cd Talib
pip install -r requirements.txt
```

Либо с указанием полного пути:

```bash
pip install -r c:\Users\Aushota\Desktop\tokenier\Talib\requirements.txt
```

---

## Запуск GUI

**Обязательно** находиться в каталоге **Talib** (или добавить его в `PYTHONPATH`):

```bash
cd c:\Users\Aushota\Desktop\tokenier\Talib
python run_app.py
```

После запуска откроется окно с вкладками: **Данные**, **Обучение**, **Бэктест**, **Предсказание**, **Оценка**.

---

## Проверка пайплайна без GUI

Скрипт проверяет цепочку: данные → признаки → окна → разбиение → обучение → бэктест → предсказание (без реальной загрузки с биржи, на синтетических данных):

```bash
cd Talib
python test_pipeline.py
```

Ожидаемый вывод в конце: `Pipeline OK`.

---

## Запуск из другой директории

Если нужно запускать не из папки Talib, задайте рабочую директорию или путь к модулям:

```bash
cd c:\Users\Aushota\Desktop\tokenier
python Talib/run_app.py
```

В `run_app.py` папка Talib добавляется в `sys.path` автоматически по пути к скрипту, поэтому такой вызов тоже допустим.

---

## Основные команды (сводка)

| Действие              | Команда                |
|-----------------------|------------------------|
| Установка зависимостей | `pip install -r requirements.txt` |
| Запуск приложения     | `python run_app.py`    |
| Тест пайплайна        | `python test_pipeline.py` |

Все команды выполнять из каталога **Talib**.
