# Talib — обучение и тест торговой модели

Приложение для обучения, тестирования и использования модели классификации ранга (направления цены) на основе признаков TA-Lib и окна OHLCV. Соответствует спецификации из документа `../data/trading_system_classifier_talib.md`.

## Возможности

- Загрузка рыночных данных через **yfinance** (тикер, период, интервал) или из CSV.
- Расчёт признаков **TA-Lib** (RSI, MACD, BBANDS, ATR, OBV и др.) и построение датасета с окнами и метками по квантилям форвард-доходности.
- Обучение классификатора (Random Forest, SVM, XGBoost) с временным разбиением и сохранение модели в joblib.
- Бэктест с учётом комиссии и проскальзывания; метрики PnL, Sharpe, max drawdown.
- Предсказание ранга по текущему окну и отображение метрик классификации и бэктеста.

## Установка

```bash
cd Talib
pip install -r requirements.txt
```

**TA-Lib:** приложение использует библиотеку `TA-Lib` для технических индикаторов. На Windows может потребоваться установка C-библиотеки TA-Lib и установка Python-обёртки (например, wheel с [unofficial Windows binaries](https://www.lfd.uci.edu/~gohlke/pythonlibs/#ta-lib) или через conda). Если TA-Lib недоступен, в коде предусмотрен запасной вариант с базовыми признаками без TA-Lib (см. `data/features.py`).

## Запуск

```bash
python run_app.py
```

Откроется окно с вкладками: **Данные**, **Обучение**, **Бэктест**, **Предсказание**, **Оценка**.
