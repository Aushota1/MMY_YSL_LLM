# Интерфейс для обучения Embedding Layer

## Описание

Интерактивный интерфейс для обучения эмбеддингов с красивой визуализацией изменений векторов в реальном времени.

## Возможности

✅ **Интерактивное меню** - удобный выбор параметров
✅ **Визуализация в реальном времени** - наблюдение за изменением эмбеддингов
✅ **График Loss** - отслеживание процесса обучения
✅ **Настройка параметров** - полный контроль над обучением
✅ **Загрузка из файла** - обучение на больших корпусах
✅ **Сохранение/загрузка моделей** - работа с чекпоинтами

## Установка зависимостей

```bash
pip install torch numpy matplotlib scikit-learn scipy
```

**Минимальные требования:**
- PyTorch
- NumPy
- Matplotlib
- scikit-learn (для t-SNE и PCA)

**Опционально:**
- scipy (для улучшенной визуализации)

## Запуск

```bash
python EMBEDDING_LAYER/embedding_trainer.py
```

## Использование

### Шаг 1: Загрузка токенизатора

1. Выберите `1` - Загрузить токенизатор
2. Введите путь к файлу (по умолчанию `chekpoint.pkl`)

### Шаг 2: Создание модели

1. Выберите `2` - Создать модель
2. Настройте параметры:
   - Размерность эмбеддингов (256, 512, 768 и т.д.)
   - Максимальная длина последовательности (512, 1024 и т.д.)
   - Скрытая размерность
   - Тип позиционного кодирования (синусоидальное/обучаемое)
   - Layer Normalization

### Шаг 3: Настройка параметров обучения

1. Выберите `3` - Настроить параметры обучения
2. Установите:
   - Количество эпох
   - Размер батча
   - Learning rate
   - Максимальная длина
   - Визуализация (включить/выключить)
   - Интервал визуализации (каждые N эпох)

### Шаг 4: Начало обучения

1. Выберите `4` - Начать обучение
2. Выберите источник данных:
   - Ввод вручную
   - Из файла
3. Введите тексты или путь к файлу
4. Наблюдайте за обучением в реальном времени!

## Визуализация

### График эмбеддингов

- **t-SNE** или **PCA** для снижения размерности
- Цветовая карта показывает близость токенов
- Подписи для важных токенов
- Обновление в реальном времени

### График Loss

- Линия показывает изменение loss
- Точки отмечают каждую эпоху
- Тренд показывает направление обучения
- Обновление в реальном времени

## Параметры обучения

### Рекомендуемые значения

**Для небольших моделей:**
- Embedding dim: 256
- Hidden dim: 256
- Batch size: 16-32
- Learning rate: 0.001
- Epochs: 10-20

**Для средних моделей:**
- Embedding dim: 512
- Hidden dim: 512
- Batch size: 32-64
- Learning rate: 0.0005
- Epochs: 20-50

**Для больших моделей:**
- Embedding dim: 768
- Hidden dim: 768
- Batch size: 64-128
- Learning rate: 0.0001
- Epochs: 50-100

## Визуализация в реальном времени

### Что показывается:

1. **График эмбеддингов:**
   - 2D проекция эмбеддингов (t-SNE или PCA)
   - Цветовая карта показывает близость токенов
   - Подписи для важных токенов
   - Обновляется каждые N эпох

2. **График Loss:**
   - Линия показывает изменение loss
   - Точки отмечают каждую эпоху
   - Тренд показывает направление
   - Обновляется после каждой эпохи

### Настройка визуализации

- **Интервал визуализации**: каждые N эпох (по умолчанию 1)
- **Метод**: t-SNE (медленнее, но лучше) или PCA (быстрее)
- **Количество токенов**: сколько токенов визуализировать (по умолчанию 50)

## Советы по использованию

### Для быстрого тестирования:

- Используйте небольшой embedding_dim (128-256)
- Уменьшите количество эпох (5-10)
- Используйте PCA вместо t-SNE
- Увеличьте интервал визуализации (каждые 2-3 эпохи)

### Для качественного обучения:

- Используйте embedding_dim 512-768
- Увеличьте количество эпох (20-50)
- Используйте t-SNE для лучшей визуализации
- Визуализируйте каждую эпоху (интервал = 1)

### Для больших корпусов:

- Загружайте тексты из файла
- Увеличьте batch_size
- Используйте GPU (если доступно)
- Уменьшите частоту визуализации

## Решение проблем

### Проблема: "ModuleNotFoundError: No module named 'sklearn'"

**Решение:**
```bash
pip install scikit-learn
```

### Проблема: Визуализация не обновляется

**Решение:**
- Убедитесь, что matplotlib в интерактивном режиме
- Проверьте, что интервал визуализации = 1
- Попробуйте уменьшить количество токенов для визуализации

### Проблема: t-SNE работает медленно

**Решение:**
- Используйте PCA вместо t-SNE
- Уменьшите количество токенов для визуализации
- Увеличьте интервал визуализации

### Проблема: Не хватает памяти

**Решение:**
- Уменьшите batch_size
- Уменьшите max_length
- Уменьшите embedding_dim
- Используйте CPU вместо GPU
