# Data Directory

Эта папка содержит текстовые данные для обучения моделей.

## Файлы

- `*.txt` - Текстовые файлы с данными для обучения
- `war_and_peace.ru.txt` - Большой текстовый корпус
- `data_1.txt`, `data_2.txt` - Датсеты для обучения
- `system_*.txt` - Системные логи и данные

## Использование

Эти файлы используются в:
- `train_gpt.py` - для обучения GPT модели
- `train_trm.py` - для обучения TRM модели
- `tokenizer_trainer.py` - для обучения токенизатора
